# 🏆 Prode Mundial 2026

Seguimiento de pronósticos del Mundial 2026 (USA · Canadá · México) para **80 participantes** y **72 partidos**.

## 🌐 Ver online
👉 [Abrir Prode](https://[tu-usuario].github.io/prode-mundial-2026/)

## ⚙️ Cómo actualizar resultados

1. Abrí el archivo `index.html` en tu navegador
2. Tocá los botones **1 / X / 2** de cada partido jugado
3. Presioná **⬇ Guardar HTML** para descargar la versión con los resultados guardados
4. Reemplazá `index.html` en este repositorio con el archivo descargado
5. Hacé commit → los cambios se publican automáticamente en GitHub Pages

## 📱 Vistas
- **⚽ Resultados** — Cargá el resultado de cada partido
- **🏅 Tabla** — Ranking en tiempo real de los 80 participantes
- **📊 Pronós.** — Tabla completa de pronósticos de todos los participantes

## 📋 Cómo crear el GitHub Pages

1. Creá un repo nuevo en GitHub (ej: `prode-mundial-2026`)
2. Subí `index.html`
3. Andá a **Settings → Pages → Branch: main / root → Save**
4. En ~1 minuto vas a tener la URL pública
